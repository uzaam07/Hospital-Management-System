# 🏥 Hospital Management System

A simple and responsive Hospital Management System developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap. The project helps hospitals manage patients, doctors, appointments, prescriptions, and billing through an easy-to-use web interface.

---

## 📖 Overview

Managing hospital records manually can be slow and prone to errors. This project digitizes everyday hospital activities such as patient registration, doctor scheduling, appointment booking, prescription management, and bill generation.

The system supports three different user roles:

- 👨‍⚕️ Doctor
- 🧑 Patient
- 👨‍💼 Administrator

Each user has their own dashboard with different permissions and features.
